Installation and Usage

  1. Download from npm:

    npm i @ah99/ngx-date-picker

  2. Import NgxDatePickerModule to your module: 

            
  import {{'{'}} NgxDatePickerModule {{'}'}} from '@ah99/ngx-date-picker'

  .
  .
  .

  @NgModule({{'{'}}

    imports: [
      NgxDatePickerModule.forRoot(),
      ...
    ]
  {{'}'}})

  3. Import css styles to your global styles: 

      
  @import "@ah99/ngx-date-picker/styles"

  4. use ngx-date-picker directive on any html input element you need:

    you need to setup one form model in your input element like formControl 

      
  {{"<"}}input ngx-date-picker [formControl]="dateControl"{{">"}}
